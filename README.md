# 💰 Expense Tracker Application

## Overview
A secure, full-stack financial tracking application designed to provide users with deep insights into their spending habits. This repository hosts the **Live Front-End Prototype** of the system, demonstrating the real-time cost categorization and dynamic budget visualization features.

## Role
**Software Developer**

## Tech Stack
* **Backend Framework (Production):** Java, Spring Boot
* **Database (Production):** SQL
* **Frontend UI (This Prototype):** HTML5, CSS3, Vanilla JavaScript
* **Data Visualization:** Chart.js

## Key Features
* **Real-Time Cost Categorization:** Automatically groups transactions into distinct categories (e.g., Food, Utilities, Transport) using logic mirroring the SQL backend.
* **Dynamic Budget Visualization:** Utilizes Chart.js to render interactive, responsive doughnut charts that instantly update as new transactions are recorded.
* **Financial Dashboard:** A comprehensive UI that calculates and displays Total Balance, Total Income, and Total Expenses instantaneously.
* **Responsive Design:** Built with CSS Flexbox and Grid to ensure a seamless experience across desktop environments.
